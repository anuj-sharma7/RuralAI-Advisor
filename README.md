# RuralAI Advisor

AI-powered, multilingual, offline-capable rural ecosystem platform built on AWS.

## Problem
Smallholder farmers face crop loss, market exploitation, and limited financial access.

## Solution
RuralAI Advisor integrates AI, ML, voice technology, and AWS cloud services to improve productivity, income, and sustainability.

## Tech Stack
- AWS Lambda
- API Gateway
- SageMaker
- DynamoDB
- S3
- Amazon Bedrock
- AWS IoT
- Amazon Lex

## Impact
- 15% Yield Increase
- 30% Income Boost
- 20% Cost Reduction
